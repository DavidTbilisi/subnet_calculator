IP Classes ( IPv4 )

IP addresses are divided into five classes: A, B, C, D, and E. Each class has a range

A, B, C - Unicast

D - Multicast

E - Experimental

Class A (Network: 1-126)

00000000.00000000.00000000.00000000

Class A IP addresses range from 00000000 to 01111111 in the first octet (X.x.x.x). The remaining three octets can be used for host addresses.

Class A IP addresses are used for very large networks, such as a major international corporation might have.

Exceptions

127 is reserved for loopback and is not a valid IP address.

0 is reserved for default route and is not a valid IP address.

Examples

10.1.1.1

Class B (Network: 128-191)

00000000.00000000.00000000.00000000

Class B IP addresses range from 10000000 to 10111111 in the first octet (X.x.x.x). The remaining two octets can be used for host addresses.

Class B IP addresses are used for medium-sized networks, such as a university campus might have.

Examples

172.16.1.1

Class C (Network: 192-223)

00000000.00000000.00000000.00000000

Class C IP addresses range from 11000000 to 11011111 in the first octet (X.x.x.x). The remaining octet can be used for host addresses.

Class C IP addresses are used for small networks, such as a small business or a home network might have.

Examples

192.168.1.1

Class D (Multicast: 224-239)

Class D IP addresses range from 11100000 to 11101111 in the first octet (X.x.x.x). These addresses are reserved for multicast groups.

Multicast addresses are used to send one packet to many hosts. Multicast addresses are used in routing protocols and other network protocols, and are not routable over the public Internet.

Examples

224.0.0.1

Class E (Experimental: 240-255)

Class E IP addresses range from 11110000 to 11111111 in the first octet (X.x.x.x). These addresses are reserved for experimental use.

Class E IP addresses are reserved for future use, or for research purposes.

Examples

240.0.0.1
